¿Cómo puedo usar console.log y console.error en JavaScript para imprimir mensajes en la consola?

Para qué sirve

Utilizar console.log y console.error en JavaScript es crucial para el proceso de depuración y desarrollo. console.log se utiliza para imprimir mensajes "normales" o información de depuración, ayudando a rastrear el flujo del programa y el estado de las variables. Por otro lado, console.error es útil para imprimir errores o advertencias, proporcionando una forma clara de identificar problemas, ya que estos mensajes suelen destacarse en la consola.
