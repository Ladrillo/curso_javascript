Acceder a la longitud y al último elemento de un array

Input ChatGPT

¿Cómo se puede acceder a la longitud de un array y a sus últimos elementos en JavaScript?

Para qué sirve

Conocer la longitud de un array y poder acceder a sus últimos elementos es crucial para el manejo de datos en JavaScript. La propiedad .length y el método .at() proporcionan maneras eficientes de interactuar con arrays, especialmente en situaciones donde el tamaño del array varía o el interés está en los elementos finales.
