¿Cómo se utiliza la etiqueta <script> para incluir JavaScript en una página web?

Para qué sirve

La etiqueta <script> es esencial en el desarrollo web con JavaScript, ya que permite integrar código JavaScript dentro de documentos HTML. Su uso es crucial para añadir interactividad y funcionalidad a las páginas web. Por ejemplo, se puede utilizar para validar datos de un formulario, actualizar dinámicamente el contenido de la página, o incluso para crear juegos y animaciones interactivas. La etiqueta <script> puede contener código JavaScript directamente o hacer referencia a un archivo JavaScript externo, lo que facilita la organización y mantenimiento del código.
