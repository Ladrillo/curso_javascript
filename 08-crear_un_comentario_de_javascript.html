Crear un comentario de JavaScript

Input ChatGPT

¿Cómo se crean comentarios en JavaScript y qué tipos de comentarios existen?

Para qué sirve

Los comentarios en JavaScript son útiles para explicar y documentar el código. Ayudan a otros desarrolladores a entender la finalidad y el funcionamiento de las partes específicas del código. Los comentarios también son útiles para desactivar temporalmente ciertas líneas de código sin eliminarlas, facilitando la depuración y el desarrollo. Existen dos tipos: comentarios de línea única y comentarios de múltiples líneas.

Ejercicios

  • Escribe un comentario de línea única que diga "Este es un comentario".
  • Escribe un comentario de múltiples líneas que cubra tres líneas, cada una con una parte de una oración.
  • Utiliza los atajos de teclado en Windows (Ctrl + /) para comentar y luego descomentar una línea de código.
