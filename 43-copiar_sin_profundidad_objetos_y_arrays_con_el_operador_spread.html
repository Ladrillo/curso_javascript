Copiar sin profundidad objetos y arrays con el operador spread

Input ChatGPT

¿Cómo se utiliza el operador spread para realizar copias superficiales de objetos y arrays en JavaScript?

Para qué sirve

El operador spread (...) en JavaScript es una forma eficiente de copiar los elementos de un objeto o array en otro nuevo. Esto es útil para crear copias rápidas y modificar o extender estructuras de datos sin alterar los originales. Sin embargo, es importante recordar que estas copias son superficiales y no clonan objetos anidados o subarrays de manera profunda.
