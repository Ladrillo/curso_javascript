¿Qué son las expresiones de funciones en JavaScript y cómo se pueden asignar a variables?

Para qué sirve

Las expresiones de funciones en JavaScript permiten asignar una función a una variable, lo cual brinda flexibilidad en la forma de definir y utilizar funciones. Esta característica es útil para crear funciones anónimas, asignar funciones como argumentos a otras funciones, y organizar mejor el código. Por ejemplo, se puede crear una función que actúe como invocadora de otras funciones, lo que es particularmente útil en patrones de diseño como callbacks y programación funcional.
