¿Cómo puedo crear rápidamente un documento HTML básico utilizando Emmet en VSCode?

Para qué sirve

Crear un documento HTML 5 es el primer paso para desarrollar cualquier sitio web. Utilizar Emmet en VSCode facilita y agiliza este proceso. Emmet es un conjunto de herramientas de asistencia para escribir HTML y CSS de manera más eficiente. Al empezar con un documento HTML básico, los desarrolladores pueden estructurar su página web y luego agregar contenido y funcionalidades específicas. La rapidez y simplicidad que ofrece Emmet lo convierten en una herramienta valiosa para cualquier desarrollador web, especialmente para principiantes que están aprendiendo las bases del desarrollo web.
