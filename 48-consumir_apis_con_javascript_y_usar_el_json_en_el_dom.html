Consumir APIs con JavaScript y usar el JSON en el DOM

Input ChatGPT

¿Cómo se puede consumir APIs en JavaScript y manipular los datos recibidos, especialmente en formato JSON, para usarlos en el DOM?

Para qué sirve

Consumir APIs y manipular datos JSON es fundamental en el desarrollo web moderno. Permite a los desarrolladores obtener y enviar datos a servicios externos, integrar funcionalidades diversas y dinamizar las páginas web. La manipulación del DOM con estos datos hace que las páginas sean interactivas y actualizadas con la información más reciente sin necesidad de recargar la página.
