Usar los métodos de arrays: map, filter, forEach

Input ChatGPT

¿Cómo se utilizan los métodos map, filter y forEach en JavaScript para manipular arrays?

Para qué sirve

Estos métodos son herramientas poderosas para trabajar con arrays en JavaScript. map se utiliza para transformar cada elemento de un array, filter para seleccionar elementos específicos según una condición, y forEach para ejecutar una función en cada elemento del array. Son esenciales para la manipulación de datos y muy útiles en tareas como procesamiento de listas, análisis de datos y operaciones de transformación.
